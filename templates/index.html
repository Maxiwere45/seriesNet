SerieNet API - Documentation

Documentation de l'API SerieNet


Cette API permet de faire de la recherche de séries TV en utilisant un algorithme de recherche par similarité.
Elle utilise aussi l'algorithme TF-IDF pour calculer la pertinence des résultats.


Les paramètres de la requête doivent être envoyés en URL Encoded.

Endpoints disponibles :

  • /search

NOTE: Cette API est en cours de développement, les endpoints peuvent changer.
Elle est principalement utilisée pour le projet SerieNet

Exemple de requête :

        
            {
                "id": "vampire",
                "lang": "VF"
            }
